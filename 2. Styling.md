# Untuk Styling di HTML kita bisa menggunakan tag `<style>` atau file CSS eksternal. Berikut adalah contoh penggunaan tag `<style>` untuk memberikan styling pada elemen HTML:

```html
<!DOCTYPE html>
<html>
  <head>
    <style>
      body {
        background-color: lightblue;
      }
      h1 {
        color: white;
        text-align: center;
      }
      p {
        font-family: verdana;
        font-size: 20px;
      }
    </style>
  </head>
  <body>
    <h1>Ini adalah heading</h1>
    <p>Ini adalah paragraf dengan styling.</p>
  </body>
</html>
```

## Di dalam style kita memiliki selector yang digunakan untuk memilih elemen HTML yang akan diberikan styling. Contohnya pada contoh di atas, `body`, `h1`, dan `p` merupakan selector yang digunakan untuk memilih elemen HTML yang akan diberikan styling.

## Setelah selector, kita memiliki deklarasi yang terdiri dari properti dan nilai. Properti adalah nama dari atribut yang ingin kita ubah, sedangkan nilai adalah nilai yang ingin kita berikan pada properti tersebut. Contohnya pada contoh di atas, `background-color`, `color`, `text-align`, `font-family`, dan `font-size` merupakan properti, sedangkan `lightblue`, `white`, `center`, `verdana`, dan `20px` merupakan nilai dari properti tersebut.

## Beberapa selector yang umum digunakan dalam CSS adalah:

- `*` untuk memilih semua elemen
- `element` untuk memilih elemen tertentu, misalnya `p` untuk memilih semua paragraf
- `.class` untuk memilih elemen dengan class tertentu, misalnya `.container` untuk memilih semua elemen dengan class "container"
- `#id` untuk memilih elemen dengan id tertentu, misalnya `#header` untuk memilih elemen dengan id "header"

## Contoh penggunaan selector

### Menggunakan selector `*` untuk memberikan styling pada semua elemen

```html
<!DOCTYPE html>
<html>
  <head>
    <style>
      * {
        color: red;
      }
    </style>
  </head>
  <body>
    <h1>Ini adalah heading</h1>
    <p>Ini adalah paragraf dengan styling.</p>
  </body>
</html>
```

Semua elemen dalam dokumen akan memiliki warna teks merah.

<br/>
<br/>
<br/>

### Menggunakan selector `element` untuk memberikan styling pada elemen tertentu

```html
<!DOCTYPE html>
<html>
  <head>
    <style>
      p {
        color: blue;
      }
    </style>
  </head>
  <body>
    <h1>Ini adalah heading</h1>
    <p>Ini adalah paragraf dengan styling.</p>
  </body>
</html>
```

Hanya elemen `<p>` yang akan memiliki warna teks biru, sedangkan elemen `<h1>` tetap menggunakan warna default.

<br/>
<br/>
<br/>

### Menggunakan selector `.class` untuk memberikan styling pada elemen dengan class tertentu

```html
<!DOCTYPE html>
<html>
  <head>
    <style>
      .highlight {
        background-color: yellow;
      }
    </style>
  </head>
  <body>
    <h1 class="highlight">Ini adalah heading</h1>
    <p>Ini adalah paragraf dengan styling.</p>
  </body>
</html>
```

Hanya elemen dengan class "highlight" yang akan memiliki latar belakang kuning, dalam hal ini adalah elemen `<h1>`.

Class cocok digunakan untuk memberikan styling yang berulang pada beberapa elemen.

<br/>
<br/>
<br/>

### Menggunakan selector `#id` untuk memberikan styling pada elemen dengan id tertentu

```html
<!DOCTYPE html>
<html>
  <head>
    <style>
      #main {
        font-size: 24px;
      }
    </style>
  </head>
  <body>
    <h1 id="main">Ini adalah heading</h1>
    <p>Ini adalah paragraf dengan styling.</p>
  </body>
</html>
```

Hanya elemen dengan id "main" yang akan memiliki ukuran font 24px, dalam hal ini adalah elemen `<h1>`.
