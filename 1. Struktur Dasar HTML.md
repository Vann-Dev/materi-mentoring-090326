# Untuk membuat struktur HTML otomatis kamu bisa ketik `!` lalu tekan `Tab` pada keyboard kamu, maka akan muncul struktur HTML otomatis seperti berikut:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
  </head>
  <body></body>
</html>
```

## Dalam HTML <...> merupakan tag, sedangkan `<!DOCTYPE html>` merupakan deklarasi tipe dokumen yang memberitahu browser bahwa dokumen ini menggunakan HTML5.

# Tag

Tag memiliki tag pembuka dan tag penutup. Tag pembuka ditulis dengan format `<nama_tag>`, sedangkan tag penutup ditulis dengan format `</nama_tag>`.

Contohnya pada tag `<p>`, tag pembukanya adalah `<p>` dan tag penutupnya adalah `</p>`.

Didalam tag terdapat **properti atau attribute** dan nilai yang disebut dengan **value**. Contohnya pada tag `<p align="center">`, `align` merupakan attribute dan `center` merupakan value dari attribute tersebut. Attribute dan value digunakan untuk memberikan informasi tambahan pada tag HTML.

```html
<p align="center">Ini adalah paragraf yang rata tengah.</p>
```

# 1. List

List merupakan tag yang digunakan untuk membuat daftar. Terdapat dua jenis list yaitu ordered list dan unordered list.

## 1.1 Ordered List

Ordered list merupakan list yang menggunakan angka atau huruf untuk menandai setiap item dalam daftar.

```html
<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>
```

LIst memiliki beberapa jenis yaitu:

- `<ol type="1">` untuk menggunakan angka (default)
- `<ol type="A">` untuk menggunakan huruf kapital
- `<ol type="a">` untuk menggunakan huruf kecil
- `<ol type="I">` untuk menggunakan angka romawi kapital
- `<ol type="i">` untuk menggunakan angka romawi kecil

## 1.2 Unordered List

Unordered list merupakan list yang menggunakan simbol seperti titik atau lingkaran untuk menandai setiap item dalam daftar.

```html
<ul>
  <li>Item A</li>
  <li>Item B</li>
  <li>Item C</li>
</ul>
```

Unordered list memiliki beberapa jenis yaitu:

- `<ul type="disc">` untuk menggunakan simbol titik (default)
- `<ul type="circle">` untuk menggunakan simbol lingkaran
- `<ul type="square">` untuk menggunakan simbol kotak
- `<ul type="none">` untuk tidak menggunakan simbol

## 1.3 Description List

Description list merupakan list yang digunakan untuk membuat daftar dengan format deskripsi. Terdapat tiga tag yang digunakan dalam description list yaitu `<dl>`, `<dt>`, dan `<dd>`.

```html
<dl>
  <dt>HTML</dt>
  <dd>Hypertext Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
  <dt>JavaScript</dt>
  <dd>Bahasa pemrograman untuk membuat website interaktif</dd>
</dl>
```

# 2. Heading

Heading merupakan tag yang digunakan untuk membuat judul atau subjudul dalam sebuah halaman web. Terdapat enam jenis heading yaitu `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, dan `<h6>`.

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

# 3. Style Text

Terdapat beberapa tag yang digunakan untuk memberikan style pada teks, diantaranya:

- `<b>` untuk membuat teks menjadi tebal (bold)
- `<i>` untuk membuat teks menjadi miring (italic)
- `<u>` untuk membuat teks menjadi bergaris bawah (underline)
- `<s>` untuk membuat teks menjadi coret (strikethrough)
- `<mark>` untuk memberikan highlight pada teks
- `<small>` untuk membuat teks menjadi kecil
- `<strong>` untuk memberikan penekanan pada teks
- `<em>` untuk memberikan penekanan pada teks dengan cara membuatnya menjadi miring (italic)
- `<sub>` untuk membuat teks menjadi subscript
- `<sup>` untuk membuat teks menjadi superscript

# 4. Menyisipkan Gambar

Untuk menyisipkan gambar dalam sebuah halaman web, kita dapat menggunakan tag `<img>`. Tag ini memiliki beberapa atribut yang penting, diantaranya:

- `src` untuk menentukan sumber gambar
- `alt` untuk memberikan deskripsi alternatif jika gambar tidak dapat ditampilkan
- `width` untuk menentukan lebar gambar
- `height` untuk menentukan tinggi gambar
  Contoh penggunaan tag `<img>`:

```html
<img src="gambar.jpg" alt="Deskripsi gambar" width="500" height="300" />
```

Dalam contoh di atas, gambar dengan nama "gambar.jpg" akan disisipkan ke dalam halaman web dengan lebar 500 piksel dan tinggi 300 piksel. Jika gambar tidak dapat ditampilkan, maka deskripsi "Deskripsi gambar" akan ditampilkan sebagai alternatif.

# 5. Menyisipkan Video

Untuk menyisipkan video dalam sebuah halaman web, kita dapat menggunakan tag `<video>`. Tag
ini memiliki beberapa atribut yang penting, diantaranya:

- `src` untuk menentukan sumber video
- `controls` untuk menampilkan kontrol pemutar video
- `autoplay` untuk memutar video secara otomatis
- `loop` untuk memutar video secara berulang
- `muted` untuk mematikan suara video

Contoh penggunaan tag `<video>`:

```html
<video src="video.mp4" controls autoplay loop muted></video>
```

Dalam contoh di atas, video dengan nama "video.mp4" akan disisipkan ke dalam halaman web dengan kontrol pemutar video, diputar secara otomatis, diputar secara berulang, dan suaranya dimatikan.

# 7. Menyisipkan Link

Untuk menyisipkan link dalam sebuah halaman web, kita dapat menggunakan tag `<a>`. Tag ini memiliki beberapa atribut yang penting, diantaranya:

- `href` untuk menentukan tujuan link
- `target` untuk menentukan bagaimana link akan dibuka (misalnya `_blank` untuk membuka di tab baru)

Contoh penggunaan tag `<a>`:

```html
<a href="https://www.google.com" target="_blank">Kunjungi Google</a>
```
